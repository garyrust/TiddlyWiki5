Review & Confirm Actions (Rsync Snapshots Only)

Files to be restored/deleted are displayed for review.

Enter a file name or path to search for a particular file

Select an action from dropdown to list files with that action

Click Next after reviewing the actions, or click Close to cancel the restore.

Restore Summary

The next page displays a summary of devices on which data will be modified. This is the final page before
restore is executed.

Read the warning and disclaimer and click Next to start the restore, or click Close to cancel.
