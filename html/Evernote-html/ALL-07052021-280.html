Oven

The oven that bakes the pizza is like the Docker platform. You
installed the oven into your house when you moved in so you could
make things with it. Similarly, you installed Docker onto your
computer so you could cook up containers.

You start your oven by turning a knob. The 

docker run

image_name

command is like your knob — it creates and starts your

container.
The cooked pizza is like a Docker container.
Eating pizza is like using your app.

Like making a pizza, making an app in a Docker container takes some
work, but in the end, you have something great. 

Wrap

make sure you don’t miss it!

Hopefully, this overview has helped you better understand the Docker

metaphors in understanding new technologies.
