If you need to configure grub-pc (for example, after an update), installing grub to all devices will break GRUB.
You will need to install to 

/dev/sda

 not 

/dev/sda1

. Installing everywhere will break the bootloader.

On a fresh install of Pop!_OS 18.04, systemd-boot is used rather than the GRUB bootloader, and the following
instructions do not apply please refer to the systemd-boot section on this page.

 is listed under
