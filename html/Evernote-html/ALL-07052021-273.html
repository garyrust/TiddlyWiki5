Using DokuWiki on smartphones

I started using DokuWiki some days ago. The first thing (after choosing and installing a template) was to get the wiki look good on my android mobile phone and when I am
browsing on my desktop PC.

I looked around here without finding a simple solution. But after investigating a bit deeper I set up a solution that works good for me. It is just an assembly of some plugins:

a second template for mobile devices (I choosed the 

Dokumobile Template

)

Loadskin Plugin

 to change the template on the fly

After installing them (all easily installable using the Extension Manager), my DokuWiki was mobile ready!
