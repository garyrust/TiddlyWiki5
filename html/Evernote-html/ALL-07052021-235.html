Editor toolbar can be customized. You can hide/show buttons you
want to use. Start by right clicking at free space in the toolbar.

Most of the buttons show usage hints and keyboard shortcuts.

Further the appearance can be adjusted using the View menu. I prefer
concise display with "Narrow Note List" and only one column (Title)
and I also mostly switch of the left sidebar.

Preferences

Further adjustments can be done in menu Edit/Preferences.

E.g. you can change UI font and also adjust note default font e.g. if
you have high resolution monitor.
