creation of desktop files for applications inside the chroot to run them with file associations and the GUI starter menu of the

host system 

> xchroot createstartup /dst/buster/usr/share/applications/xine.desktop

 

source file: /dst/buster/var/lib/xine/xine.desktop
destination file: /usr/share/applications/xine-buster.desktop
root: /dst/buster/
xchroot: /home/shell/shutil/xchroot
icon: /dst/buster/usr/share/icons/hicolor/48x48/apps/xine.png

current working directory is preserved by the xchroot/openroot call if it exists inside the chroot
This is especially useful if you want to mount the home directory inside the chroot:

> grep /dst/buster/home /etc/fstab

/home                     /dst/buster/home      none    bind      0 0
