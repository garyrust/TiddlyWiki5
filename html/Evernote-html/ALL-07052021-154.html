Inserts a bookmark, a handle that follows the text around it as it is

find()

By default, text typed when the cursor is on top of the bookmark
will end up to the right of the bookmark. Set this option to true to
make it go to the left instead.

See the corresponding 

option

 to

As with 

markText

widget inserted for this bookmark are handled by CodeMirror. The
default is false.
