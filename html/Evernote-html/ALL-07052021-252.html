21 February 2015: i486 architecture removal from PLD Th

On 28 February 2015 i486 architecture will be removed from PLD Th main tree.

This also means drop of support for i586 class CPUs and some old, rare, “degraded” i686 chips lacking cmov instructions support.

Growing number of packages that either do not build for i486 or are severly crippled (like IcedTea or programs requiring atomic
operations) with loss of users of this architecture caused us to finally abandon it.

i486 will be removed from main/ready/test trees, It will still be available in archived yearly snapshots.

8th July 2014: Kernel packaging news for PLD Th

Kernel line 3.14 has been announced as new longterm and, because of this, we will now ship three longterm kernel package sets

falling behind the mainline kernel and only really supported for the stable/longterm lines.
