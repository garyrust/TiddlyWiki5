Commands

Commands are parameter-less actions that can be performed on an
editor. Their main use is for key bindings. Commands are defined by
adding properties to the 

CodeMirror.commands

 object. A number of

common commands are defined by the library itself, most of them used
by the default key bindings. The value of a command property must be a
function of one argument (an editor instance).

Some of the commands below are referenced in the default key map, but
not defined by the core library. These are intended to be defined by user
code or addons.

Commands can also be run with the 

execCommand

 method.
