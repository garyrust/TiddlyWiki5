Initial customization

As many people prefer different ways to use the app. There are some
customizations possible to adjust the default setting to your needs.
You can start by adjusting the displayed columns. Per default only
"Title" is displayed, but there are lot of other columns you want to
display.

Also you can rearrange the order by dragging the columns.
