That's a good question, and I suspect that the answer is "yes, with
enough pain". The PulseAudio goop appears to be glued via
filesystem sockets, and those can probably be transported to a less-
privileged user via a shared group. However, it's almost certainly not
as easy as just creating that group and putting both users in it, then
chowning the existing sockets. (As I recall, for one thing, there are
mktmp-like directory names in there someplace.)
