Objective

To install and configure grub boot loader for POP OS.

Prerequisite/Requirements

POP OS 20.04 Installed
User account with sudo access

Warning

After installing and initializing grub do not reboot your system until
the end

Step 1: Install necessary packages

First and foremost update and upgrade your POP OS if it is newly
installed.

Open up terminal and enter:

Command

sudo

 

apt

 update -y 

&&

 

sudo

 

apt

 upgrade -y
