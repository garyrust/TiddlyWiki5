Part 2
Using Solid Explorer File Manager

1. 

Helpful?

Open Solid Explorer File Manager. It's a blue folder app. This
app will allow you to save extracted APKs to your Android's
internal storage, from which point you can send the APKs to
another device by sharing them.

[2]

If you haven't yet downloaded Solid Explorer File Manager, first
do so from the Play Store:

https://play.google.com/store/apps/details?
id=pl.solidexplorer2&hl=en

The full version of this app costs $1.99. You'll need to purchase
the full version after fourteen days.

Helpful?
